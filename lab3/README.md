## Script run's to setup the network according to the lab guide

- Run the `initiate-nw.sh` script.
- Run the `nw-interface-setup.sh` script providing the host number.
- Run the `configure-if.sh` script on each host providing the host number.
- Run the `run-olsr-sh` script to initiate the OLSR network, on each host providing the 
host number.
- Ping host 1 from host 4 `ping6 2029::1`.
